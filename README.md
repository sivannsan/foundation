# SIVANNSAN Foundation
A Java module providing basic Java library for general use cases

### All features
+ Annotations
    - Nonnegative
    - Nonnull
+ Common
    - Check
    - Ensure
    - Validate
+ Security
    - Encryption
+ Utility
    - DateUtility
    - FileUtility

### Maven
```xml
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>
```
```xml
<dependency>
    <groupId>com.github.sivannsan</groupId>
    <artifactId>foundation</artifactId>
    <version>1.0.0</version>
</dependency>
```
